# python中的切片
1. 切片的常见用法
python中的切片,通常用法是
```python
x = [1,2,3,4,5]
x[1:4]
# [2,3,4]
```
这表示从下标1到下标3

2. 下标为负数
下标可以为负数
```pyton
x[0,-1]
# [1,2,3,4]
```
3. 下标可以省略
```python
x[:-1]
```

4. **步进**
```python
x[0:4:2]
#[1,3]
```
这表示下标从0到3,步进为2
```python
x[:3:-1]
#[5]
```
如果步进为负数,那就是从后往前进到下标为3

特殊技巧:实现反转:
```python
x[::-1]
# 前两位都缺省,步进为-1
#[5,4,3,2,1]
```
