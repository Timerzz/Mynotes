# zip
- zip() 函数用于将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的列表。

- 如果各个迭代器的元素个数不一致，则返回列表长度与最短的对象相同，利用 * 号操作符，可以将元组解压为列表。


1. zip使用
```python
a = ['a',1,3]
b = ['b',4,5]
c = zip(a,b)     # 打包为元组的列表
for i in c:
	print(i)
#('a', 'b')
#(1, 4)
#(3, 5)
```

2. 用*解压
```python
d = zip(*c)
for i in d:
	print(i)
#('a', 1, 3)
#('b', 4, 5)
```
