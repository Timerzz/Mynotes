1. 空接口可以表示任何类型,可以接受任何类型
2. 通过断言将空接口转换为指定类型
```go
v,ok := p.(int)
```

```go
func DoSomething(p interface{}){
	if i,ok:=p.(int);ok{
		fmt.Print("Integer", i)
		return
	}
	if i,ok:=p.(string);ok{
		fmt.Print("string", i)
		return
	}
```
