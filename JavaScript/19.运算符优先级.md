# 题目 
 1. ||和&&运算的题目
```js
console.log(0||1);//1
console.log(1||0);//1
console.log(3||1);//3
console.log(1||3);//1
```
&&运算符优先级大于||
```js
var a = 5 && 0 || 1;   //1
var b = 4 || 0 && 8;   //4
var c = 0 || 8 && 9;   //9
```

2. +，-和||，&&
```js
var c = 4 && 3 && 8 - 1;   //7
```
